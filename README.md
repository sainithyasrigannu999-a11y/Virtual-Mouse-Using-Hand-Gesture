# Virtual-Mouse-Using-Hand-Gesture
> This project introduces a touchless, vision-based human-computer interface that allows users to control their computer's mouse cursor dynamically using real-time hand gestures. Developed entirely in Python, the system utilizes a standard web camera to capture physical movements and translate them instantly into digital screen coordinates, eliminating the need for a physical mouse hardware device.

> Unlike conventional modern setups that rely heavily on machine learning frameworks or complex pre-built tracking pipelines like MediaPipe, this implementation relies on traditional Computer Vision techniques.
